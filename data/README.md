# 数据目录

本目录包含实验相关的所有数据文件。

## 目录结构

```
data/
├── README.md           # 本说明文件
├── raw/                # 原始数据
│   ├── experiment_1/   # 实验1原始数据
│   ├── experiment_2/   # 实验2原始数据
│   └── ...            # 其他实验数据
├── processed/          # 处理后的数据
│   ├── cleaned/        # 清洗后的数据
│   ├── normalized/     # 归一化数据
│   └── features/       # 特征数据
└── results/            # 实验结果数据
    ├── tables/         # 结果表格
    ├── figures/        # 结果图表数据
    └── logs/           # 实验日志
```

## 数据说明

### 原始数据（raw/）
- 实验设备直接采集的原始数据
- 保持数据原始状态，不做任何修改
- 包含时间戳、传感器读数等原始信息

**示例数据格式：**
```
timestamp, sensor_1, sensor_2, sensor_3
2024-01-01 10:00:00, 1.234, 5.678, 9.012
2024-01-01 10:00:01, 1.235, 5.679, 9.013
```

### 处理后数据（processed/）
- 经过预处理的数据
- 包含数据清洗、归一化、特征提取等步骤的产物
- 数据格式统一，便于后续分析

### 实验结果（results/）
- 实验运行产生的结果数据
- 包含性能指标、统计数据等
- 附带实验日志，记录实验过程

## 数据格式规范

1. **CSV格式**：使用UTF-8编码，逗号分隔
2. **JSON格式**：使用标准JSON格式，便于程序读取
3. **NumPy格式**：大型数值数据建议使用.npy或.npz格式
4. **数据命名**：使用下划线命名法，如`experiment_1_data.csv`

## 注意事项

- 原始数据禁止修改，所有处理后的数据另存
- 大型数据文件建议使用Git LFS管理
- 敏感数据请加密存储或添加到.gitignore
